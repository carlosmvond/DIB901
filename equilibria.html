\(\dfrac{dx}{dt} = f(x)\) — Análisis cualitativo y diagrama de fase

Función f(x)

Expresión inválida. Revisa la sintaxis.

Ejemplos predefinidos

Operaciones básicas
x + 2,  x - 1,  x * 3,  x / 2
x**2  o  pow(x,2)  → potencias

Funciones disponibles
sqrt(x) → raíz cuadrada
exp(x) → e^x
log(x) → logaritmo natural
abs(x) → valor absoluto
sin(x), cos(x), tan(x)
PI → π ≈ 3.14159

Ejemplos
x*(1 - x/5) → logístico
x*(3-x)*(x-1) → Allee
x*(1-x/5) + 0.3 → con inmigración
sqrt(x) - 0.5*x → raíz
x*exp(-x) - 0.1 → exponencial

Nota: usa * siempre para multiplicar.
2x no funciona — escribe 2*x.
